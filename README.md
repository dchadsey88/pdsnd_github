### Project Title
Bike Share Analysis

### Description
This project allows users to select bikeshare data from one of three cities (NYC, Chicago, DC) as well as sort the data by month or day of the week.  The program than returns a summary of the data.

### Files used
new_york_city.csv
chicago.csv
washington.csv
